ШКОЛЬНАЯ ФОРМА — ВИТРИНА (САЙТ-КАТАЛОГ)
========================================

СТРУКТУРА ПАПКИ
- index.html        — сам сайт (открывается в любом браузере)
- images/           — все фотографии
    bg.webp         — фоновое фото (закат, здание)
    m1_01..m1_04    — фото первой модели (Жилет + юбка)


КАК ПОСМОТРЕТЬ ЛОКАЛЬНО
Просто дважды кликни по index.html — откроется в браузере.
(Шрифты подгружаются из интернета, поэтому нужен интернет.)


КАК ОПУБЛИКОВАТЬ И ПОЛУЧИТЬ ССЫЛКУ ДЛЯ ТЕЛЕФОНА (GitHub Pages)
1. Зайди на github.com → New (создать репозиторий) → имя, например school-uniform
   → выбери Public → Create repository.
2. Add file → Upload files → перетащи СЮДА index.html И папку images целиком
   → внизу Commit changes.
3. Settings → Pages → Source: Deploy from a branch → ветка main, папка / (root) → Save.
4. Подожди 1-2 минуты, обнови страницу — появится ссылка вида:
   https://ТВОЙЛОГИН.github.io/school-uniform/
   Эту ссылку открывай на телефоне и кидай в WhatsApp.

Важно: репозиторий должен быть Public, иначе бесплатный GitHub Pages не работает.


КАК ВСТАВИТЬ НОМЕР WHATSAPP
В index.html найди строку со словами wa.me/0000000000
Замени 0000000000 на номер в международном формате без + и без пробелов.
Пример для России: 79991234567


КАК ДОБАВИТЬ НОВЫЕ МОДЕЛИ (позже)
1. Положи фото модели в папку images (например m2_01.webp ... m2_05.webp).
2. В index.html найди блок:  const MODELS=[ ... ]
3. Добавь модель по образцу первой:
   {id:2, title:"Название", photos:["images/m2_01.webp","images/m2_02.webp", ...]},
Фотографии лучше заранее сжимать (ширина ~1080px, формат webp) — так сайт быстрый.

Если что-то непонятно — пришли это Claude, поможем.
